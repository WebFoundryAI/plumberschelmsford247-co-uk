# Bath Plumbers

Professional plumbing website for Bath and North East Somerset.

## Overview

Static HTML/CSS website optimised for SEO and fast loading. Designed for deployment on Cloudflare Pages.

## Structure

```
/
├── index.html                    # Homepage (1000+ words)
├── emergency-plumber-bath.html   # Emergency plumber service
├── boiler-servicing-bath.html    # Boiler servicing service
├── bathroom-fitting-bath.html    # Bathroom fitting service
├── leak-detection-bath.html      # Leak detection service
├── about.html                    # About page
├── contact.html                  # Contact page
├── privacy-policy.html           # Privacy policy
├── terms.html                    # Terms of service
├── sitemap.xml                   # XML sitemap
├── robots.txt                    # Robots file
├── services/
│   └── index.html                # Services index
├── locations/
│   └── index.html                # Locations index
└── assets/
    └── css/
        └── style.css             # Main stylesheet
```

## Features

- Static HTML/CSS (no JavaScript frameworks)
- Mobile-responsive design
- SEO optimised with JSON-LD schemas
- Canonical URLs
- Internal linking structure
- UK English content
- Period property expertise content

## Schema Types

- LocalBusiness (homepage)
- FAQPage (homepage)
- Service (service pages)

## Deployment

Deploy to Cloudflare Pages connected to this repository.
