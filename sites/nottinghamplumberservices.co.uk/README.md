# Nottingham Plumber Services

Professional plumbing services website for Nottingham and Nottinghamshire.

## Structure

```
/
├── index.html                         # Homepage (1000+ words)
├── emergency-plumber-nottingham.html
├── boiler-services-nottingham.html
├── bathroom-installation-nottingham.html
├── blocked-drains-nottingham.html
├── about.html
├── contact.html
├── privacy-policy.html
├── terms.html
├── services/
│   └── index.html
├── locations/
│   └── index.html
├── assets/
│   └── css/
│       └── style.css
├── images/
├── sitemap.xml
├── robots.txt
└── README.md
```

## Technical

- Static HTML/CSS
- No JavaScript frameworks
- Responsive design
- SEO optimised with JSON-LD schemas
- Cloudflare Pages compatible

## Deployment

Connect to Cloudflare Pages with output directory `/`.

## Customisation

Update phone numbers, email addresses, and business details in all HTML files before going live.
